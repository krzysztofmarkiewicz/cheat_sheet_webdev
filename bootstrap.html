  6. MARGIN/PADDING - info o marginesach i paddingach nagetbootstrap.com oraz na UDEMY
    • klase tworzymy w taki sposób
      dla breakpointu xs {property}{sides}-{size}
      dla breakpointów wyższych sm, md, lg, and xl {property}{sides}-{breakpoint}-{size}
    • Where property is one of:

      m - for classes that set margin
      p - for classes that set padding
    • Where sides is one of:
      • t - for classes that set margin-top or padding-top
      • b - for classes that set margin-bottom or padding-bottom
      • l - for classes that set margin-left or padding-left
      • r - for classes that set margin-right or padding-right
      • x - for classes that set both *-left and *-right
      • y - for classes that set both *-top and *-bottom
      • blank - for classes that set a margin or padding on all 4 sides of the element
    • Where size is one of: (mnożnik REM - czyli jeśli w css ustawimy font size: 10px w tagu html to .25rem to jest 2.5px)
      • 0 - for classes that eliminate the margin or padding by setting it to 0
      • 1 - (by default) for classes that set the margin or padding to $spacer * .25
      • 2 - (by default) for classes that set the margin or padding to $spacer * .5
      • 3 - (by default) for classes that set the margin or padding to $spacer
      • 4 - (by default) for classes that set the margin or padding to $spacer * 1.5
      • 5 - (by default) for classes that set the margin or padding to $spacer * 3
      • auto - for classes that set the margin to auto
    • example: div class="mx-auto"
      example: div class="px-md-5" - w css przy byłoby: padding: 0 5rem; - ale tylko w breakpointcie md
    • jeśli chcemy ustawic padding lub margin chcemy nadac ujemny to przed mnoznikiem dodajemhy literkę n (od negative)
