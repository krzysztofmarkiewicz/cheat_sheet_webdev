MDN o FLexboxie
SUPER GRA DO NAUKI FLEXBOXA

Flexbox jest jednowymiarowy. Elementy ustawiamy albo na osi x albo na osi y czyli albo poziomo albo pionow. Ustawiamy je wtedy jako ROW albo COLUMN. Domyślnie ustwaiona jest opcja row. Do ustwaienia trybu row służy "flex-direction: row". Są jeszcze inne np flex-direction:row-reverse i inne

Flexbox jest świetny do wyśrodkowywania (align-items oraz justify-content)

Flexa ustwaiamy na rodzicu

flexflow - można ustawic direction i wrap w jednym wierszu

Ściąga z właściwości Flexa

FLEX CONTAINER (rodzic) :

  • display: flex / inline-flex;

  • flex-direction: row [ wartość domyślna ] / column / row-reverse / column-reverse;

  • flex-wrap: wrap / nowrap [ wartość domyślna ];

  • align-items (prostopadłe do osi głównej): stretch [ wartość domyślna ] / flex-start / flex-end / center / baseline(wyrównuje wszystkie elementy do dolnej krawdzi czcionki);

  • justify-content (równoległe do osi głównej): flex-start [ wartość domyślna ] / flex-end / center / space-around / space-between / space-evenly;

  • align-content: stretch [ wartość domyślna ] / center / flex-start / flex-end / space-around / space-between; - żeby align-content działał muszą być przynajmnej dwie linie elementów w kontenerze


FLEX ITEM (dziecko):

  • flex-basis: auto [ wartość domyślna ] / x; - określa rozmiar elementu. Przy "flex-dierction: row" odnosi się do width, a przy "flex-direction: column" do height. Ma pierwszeństwo przed właściwością width oraz właściwością height

  • flex-grow: 0 [ wartość domyślna ] / x; - określa proporcję rozrostu elementu. Element urośnie jeśli w lini jest miejsce, powięklszy się do maks rozmiaru rodzica

  • flex-shrink: 0 / 1 [ wartość domyślna ]; - określa proporcję kurczenia się elementu w stosunku do innych z tą własciwością ustawioną inaczej

  • order: 0 [ wartość domyślna ] / x;z pomocą właściwości “order” możemy dowolnie zmieniać kolejność elementów w kontenerze

  • align-self: auto [ wartość domyślna ] / stretch / flex-start / flex-end / center; działa tak jak align-items, ale działą tylko na konkretny element bo nie stosujemy go na rodzicu tylko na danym elemencie


Zapisy skrócone:

  • flex: [grow] [shrink] [flex-basis];

  • flex-flow: [flex direcrion] [flex wrap]


Przykłady:

flex: 1 2 30px

to zapis skrócony dla:

  1. flex-grow: 1;
  2. flex-shrink: 2;
  3. flex-basis: 30px;


flex-flow: row wrap

to zapis skrócony dla:

  1. flex-direction: row;
  2. flex-wrap: wrap;
